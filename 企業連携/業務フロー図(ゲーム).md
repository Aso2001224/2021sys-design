```
@startuml
opt 未登録
ユーザー -> webサーバー:ユーザー登録
webサーバー -> DBサーバー:ユーザー登録
DBサーバー -> DBサーバー:登録処理
DBサーバー->webサーバー:登録結果

alt 登録成功
webサーバー-> ユーザー:登録メッセージを表示
else 登録失敗
end

@startuml
opt ログイン
alt ログイン成功
webサーバー->ユーザー: ログイン成功画面表示
else ログイン失敗
opt ログイン中
ユーザー->webサーバー:ログアウト
webサーバー->DBサーバー:ログアウト照会
DBサーバー->DBサーバー:ログアウト処理
DBサーバー->webサーバー:ログアウト結果
webサーバー->ユーザー:ログアウト結果
end

@startuml
activate ユーザー
ユーザー->webサーバー:商品検索
webサーバー->DBサーバー:検索問い合わせ
DBサーバー->DBサーバー:検索処理
DBサーバー->webサーバー:検索結果
alt 検索成功
webサーバー->ユーザー:該当商品を表示
else 該当商品がない
webサーバー->ユーザー:該当商品なし
end

@startuml
loop 買い物
opt 買い手が別の出品を探している
webサーバー->DBサーバー:その中古品のデータを出力
else 何もしない
opt ユーザーが納得した
ユーザー->webサーバー:商品をカートへ
else ユーザー　->webサーバー:何もしない
end
